# AI Agent-Driven Task Manager

A learning project demonstrating **agent-driven development** with autonomous AI agents coordinating to plan and execute tasks.

## 🎯 Project Vision

This isn't just another todo app - it's an exploration of how AI agents can work together to solve problems autonomously. Two specialized agents collaborate:

- **Planning Agent**: Analyzes requirements, breaks down complex tasks, assigns priorities
- **Implementation Agent**: Executes tasks, generates outputs, reports progress

## 🏗️ Architecture

```
User Request → Planning Agent → Task Breakdown → Implementation Agent → Execution
```

See [AGENT_PROJECT_SUMMARY.md](AGENT_PROJECT_SUMMARY.md) for complete architecture and implementation plan.
